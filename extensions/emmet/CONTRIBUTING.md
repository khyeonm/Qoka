## How to build and run from source?

Read the basics about extension authoring from [Extending Qoka](https://code.visualstudio.com/docs/extensions/overview)

- Read [Build and Run Qoka from Source](https://github.com/microsoft/vscode/wiki/How-to-Contribute#build-and-run-from-source) to get a local dev set up running for Qoka
- Open the `extensions/emmet` folder in the vscode repo in Qoka
- Press F5 to start debugging

## Running tests

Tests for Emmet extension are run as integration tests as part of VS Code.

- Read [Build and Run Qoka from Source](https://github.com/microsoft/vscode/wiki/How-to-Contribute#build-and-run-from-source) to get a local dev set up running for Qoka
- Run `./scripts/test-integration.sh` to run all the integrations tests that include the Emmet tests.
